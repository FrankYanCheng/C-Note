数组传递的是通过指针，默认指向第一个下标指向的值，
void printingc(int ia[4])
{
	ia[2] = 100;
}
//保留的是实参的指针副本，因此可以改变所指向的数组元素的值，但是对实参的指针没有影响
void printinga( int* ia)
{
	for (size_t i = 0; i!=4; i++)
	{
		//cout << *ia << endl;
		*ia = 1;
		ia++;

	}
}
这两种形式等价

通过引用传递数组
这种方式传递的数组通过引用方式，所以在编译器检查时会检查数组大小是否合理。这种方式比较安全
必须加括号,优先和括号结合
void test(int(&arr)[10]);
